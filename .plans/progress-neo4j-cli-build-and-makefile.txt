  config, added neo4j-cli to notarize ids, upgraded format→formats, added archive IDs)

Key Decisions:
- GoReleaser v2 deprecates `archives.format` (string) — use `archives.formats` (array)
- GoReleaser v2 deprecates `format_overrides.format` — use `format_overrides.formats`
- Each archive block must have a unique `id` (otherwise defaults to "default" and
  causes a "found 2 archives with the ID 'default'" error)
- Used `{{ .Binary }}` in archive name_template (not `{{ .ProjectName }}`) so each
  binary's archives are named after the binary, not the project
- `project_name` changed from `aura-cli` to `neo4j-cli` to reflect the super-CLI
